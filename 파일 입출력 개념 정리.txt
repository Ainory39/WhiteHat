1. 파일 입출력
    1.1 FILE *
       처리할 파일을 가리킬 파일 포인터가 필요하다. 이때 FILE *자료형을 사용해 파일을 가리킬 포인터 변수를 생성한다.
       
       FILE *(파일 포인터 이름 = stream 이름);
      
    1.2 fopen()
       파일을 열때 사용하는 함수이다. 함수의 사용법은 다음과 같다.
       
            fopen( 처리하려는 파일 이름, 파일 처리 모드);
            
       파일 처리모드는 다음과 같은 것들이 있다.
       
         모드    |   기능    |   파일 없음   |    파일 있음
          r          읽기          에러        기존 파일 사용
          r+       읽기/쓰기       에러        기존 파일 사용
          w          쓰기       새로 생성         새로 생성
          w+       쓰기/읽기    새로 생성         새로 생성
          a         덧붙이기    새로 생성        기존 파일 사용
          a+       쓰기/읽기    새로 생성        기존 파일 사용
          
          
      1.3 fclose()
          fopen()으로 파일을 열고, 사용을 마쳤다면 반드시 종료를 해주어야 한다. 이때 fclose()를 이용해 파일을 닫을 수 있다.
          함수의 사용법은 다음과 같다.
          
              fclose(파일 stream 이름);
              
      1.4 파일에서 가져오기
        파일에서 값을 읽어올 때 사용할 수 있는 대표적인 함수들이다.
        
          fscanf(stream 이름, "읽어올 자료의 형식", 읽어들일 위치);
           -공백, 개행문자, EOF를 만나면 그 전까지만 입력받는다.
          fgets(읽어들일 위치, 읽어올 문자열 크기, stream 이름);
           -개행문자, EOF를 만나면 그 전까지만 입력받는다.
           
      1.5 파일에 출력하기
         fputs("쓸 내용", 쓸 stream 이름);
         
         fprintf(쓸 stream 이름, "쓸 내용", 서식 문자);
          -fprintf는 fputs와 다르게 형식 지정자를 사용할 수 있다.
          
      1.6 위치 지정자
       현재 파일에서 입력받고 출력할 위치를 지정하고 싶을 때가 있을 때 사용하는 함수는 fseek()이다.
       
           fseek(stream 이름, offset, 기준점);
           
        -offset : 기준점으로부터의 거리를 나타내는 정수
        -기준점
          ● 파일의 시작 : SEEK_SET
          ● 현재 가리키고 있는 위치 : SEEK_CUR
          ● 파일의 끝 : SEEK_END
